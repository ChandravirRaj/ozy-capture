# desktop_screenshot_capture_windows

Windows implementation of `desktop_screenshot_capture` using Windows Graphics Capture (WGC).

## Channels

- Method: `dev.oxy.screen_capture/desktop_screenshot_capture`
- Events: `dev.oxy.screen_capture/events`

## Requirements

- Windows 10 version 2004+ / Windows 11
- Visual Studio 2022 with Desktop C++ workload

See [`../../docs/windows/build.md`](../../docs/windows/build.md) for build and test instructions.
